The week in personal AI agents

Computer use goes mainstream

Google has pushed computer-use capabilities into Gemini 3.5 Flash, signaling that desktop control is becoming a baseline feature for advanced agents (memeburn.com, forbes.com).

Security pressure rises

Researchers continue to demonstrate how AI agents with environment access can become credential exfiltration paths, from MCP flaws in Amazon Q to new agent-hijacking techniques (darkreading.com, cryptobriefing.com).

Demand keeps growing

Consumer appetite for personal AI agents is rising globally, with especially strong interest across Asia-Pacific (bwbusinessworld.com).

How the landscape is shaping up

ChatGPT

A general-purpose AI assistant evolving toward agents and task automation. Excellent for ad‑hoc work, but repeated computer workflows are not its core focus.

Gemini

Google is aggressively pushing browser and computer control, making Gemini a reference point for large-scale agent deployment.

Grok

xAI’s assistant is iterating quickly, with new model versions and voice APIs landing across platforms (cryptobriefing.com).

Siri

Deeply embedded in Apple’s ecosystem and voice-first by design, but constrained in how freely it operates across the open web.

Folk

A niche player in the broader automation and CRM-adjacent agent space — relevant context, but not a computer-operating agent story.

Orchids

An experimental approach to agents and automation, representing ongoing exploration rather than a mainstream platform.

Super

Built for people who want a personal AI agent that actually operates a computer — and crucially, can reuse a computer-use cache, making repeated workflows faster and cheaper over time.

Why caching computer use matters

As agents gain the ability to click, type, browse, and configure real systems, the cost and reliability of repeated execution becomes the bottleneck. Super is positioned around this reality.
